# SquidService
## Getting started
```
git clone https://github.com/rburbidge/SquidService.git
cd SquidService
npm install
npm install -g typescript gulp
```

## Commands
* Build: ```tsc -w```
* Run the server locally: ```npm start```
* Clean: ```gulp clean```

## Deploying to the server
* Deploy types to the server: ```gulp deployTypes```
